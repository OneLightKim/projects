	팀원간 신속한 프로토타입 제작 및 공유로 반복적 접근 방식이 원칙
VCS : 문서나 설계도, 소스 코드 등의 변경점을 관리해주는 소프트웨어
	변경점 관리, 버전 관리, 백업 및 복구, 협업




스칼라 : 숫자 하나로 이루어진 데이터
벡터 : 여러 숫자로 이루어진 데이터 레코드
행렬 : 벡터가 조합된 데이터
텐서 : 같은 크기의 행렬이 조합된 데이터
퍼셉트론 : 비선형 같은 복잡한 함수 표현 가능
	컴퓨터 수행 복잡한 처리 표현 가능
	가중치는 수동
신경망 : 가중치 매개변수의 적절한 값을 데이터로부터 자동 학습

step function : 
sigmoid function : 
ReLU : x>0 => x , x<=0 =>0
activate function
-linear, sigmoid계열 : s형태, 절대값 크기 커질수록 0에 수렴, 비선형
-ReLU 계열 : 0을 기준 기울기 값이 두개로
softmax function : 분류에서 확률값으로 출력 .. max의 근사값

@NNfit(신경망 학습) : 손실 함수 결과를 가장 작게 만드는 가중치 매개변수 찾는 것이 학습 목표

Loss function : 
	MeanSquaredError(MSE) : y는 신경망 출력 값, t는 target value	
	CrossEntropyError(CEE) : classification문제시 one_hot, 원 핫 인코딩 했을 시에만 활용함.
			y가 정확히 예측했을 때 cost값이0
정확도를 바로 구하기 전 손실함수를 이용한다(미분의 역할에 중점)
	수치미분 : 작은 차분으로 미분을 구한다
	편미분 : 특정 잠수의 기울기를 구함. 
	기울기 : 모든 변수의 편미분 값을 벡터로 정리
	경사하강법 : 손실함수의 최솟값을 찾아감
	미니배치  : 6만개 훈련 시 손실함수 6만개의 평균 구해야함
		근사치로 계산도 가능함
뮤 : learning rate를 통해 ,overfitting을 조절. (너무 작으면 학습시간 길고 overfitting)
learning rate decay : 학습을 진행하며 학습률을 줄여감
AdaGrad : 변수 update시 마다 각각 변수의 step size를 다르게 설정
